PORTFOLIO

Fork

A recipe sharing app where users can post and browse recipes. Once logged in, users can explore featured recipes, look through recipe categories, and search for a specific recipe. Along with the standard ingredients and instructions, recipes also have a notes section where users can offer feedback about the recipe. If a user is inspired or want to create their own spin on a recipe, a user can fork it using a dynamic multi-page form.
